## 💻 Proyectos Destacados

### 🎯 [Sistema de Control de Asistencia con QR](https://github.com/tapiat/asistenciaqr)
**Stack:** Java • Spring Boot • Angular • Ionic • MySQL • JWT

Sistema fullstack para gestión de asistencia en tiempo real mediante códigos QR.

- ✅ API REST con arquitectura en capas (autenticación, lógica de negocio, acceso a datos)
- ✅ Autenticación JWT con control de acceso por roles sin estado en el servidor
- ✅ Aplicación móvil Ionic integrada con backend vía API REST
- ✅ Tiempos de respuesta menores a 300ms en operaciones críticas
- ✅ Base de datos relacional con 6 tablas e integridad referencial

### 🛒 Eco Market SPA - Arquitectura de Microservicios
**Stack:** Java • Spring Boot • MySQL • Maven • REST API

Sistema e-commerce con arquitectura de microservicios desacoplados.

- ✅ 4 microservicios independientes (usuarios, productos, pedidos, pagos)
- ✅ Separación de responsabilidades por dominio
- ✅ CRUDs completos con validación de datos y manejo estructurado de errores HTTP
- ✅ Resolución de problemas de comunicación entre servicios REST

### 🍔 [E-commerce Duco Burger](https://github.com/tapiat/ecomercado)
**Stack:** Node.js • Express • MySQL • JavaScript

Plataforma e-commerce para gestión de pedidos con control de inventario.

- ✅ Modelo de datos relacional con entidades de productos, usuarios y pedidos
- ✅ Sistema de roles (administrador/cliente) con control de acceso
- ✅ Validación de stock en pedidos concurrentes
- ✅ Panel de administración con gestión de pedidos e historial por usuario
